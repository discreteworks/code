## Dependencies

You need install cmake.


## Build Excercise

These instructions are for Linux machine.

Within the root directory of deitelspp-9th

```
mkdir build
cd build
cmake ..
make -j ${nproc}
```

## Run Chapter 2 Excercise

Within root directory of deitelspp-9th
